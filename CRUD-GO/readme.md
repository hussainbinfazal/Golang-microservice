## 📋 Prerequisites

- 🐹 **Go 1.21** or higher
- 🔧 **Git**

## 🛠️ Installation

1. **Clone the repository:**
```bash
git clone https://github.com/hussainbinfazal/Golang-microservice.git
cd Golang-microservice
```

2. **Install dependencies:**
```bash
go mod tidy
```

3. **Run the application:**
```bash
go run main.go
```
